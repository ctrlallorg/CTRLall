    Formatting text

    ⭐ Beginner Level

    Formatting means changing the appearance and layout of your text and document. Formatting tools are found on the Home tab of the ribbon and are there to help you make your content more readable. This article explains what each one does.

    Clipboard tools

    These tools are to do with copying and moving content such as a cut, copy and paste.

    Cut, copy and paste

    These are tools to copy and/or move content.

    • Cut: This removes the content from the page and places it on the clipboard which acts as Word's short term memory. You can then paste it elsewhere. It's a way of moving content around your document.

    • Copy: This copies the content to the clipboard so you can paste it elsewhere, letting you duplicate it.

    • Paste: This duplicates content copied to the clipboard by cutting or copying it first. There are different types of paste options depending on what you cut or copy which are explored more in the intermediate version of this article.

      • Format Painter

      This is a really useful tool which lets you apply the formatting/look of some selected text onto other text in your document. This is much faster than manually applying the same formatting each time.

      Note: If you're wanting to format headings you should use styles (explored here: Styles). With heading styles you can use the format painter to apply it to other text to make it a recognised heading in Word.

    Formatting tools change the appearance and layout of your text and document; they make your content more readable, professional, and visually appealing. They can be found on the Home tab of the ribbon. This article explains what each one does.

    Clipboard tools

    Cut, copy and paste

    These are tools to copy and/or move content.

    • Cut: see Moving content
    • Copy: see Copying content
    • Paste: depending on what content you've cut or copied you'll see different icon options, for text it's generally these four:
    Paste icon Definition
    Paste - Keep Source Formatting
    		Pastes the content bringing its inherent formatting to your document. This may not always be something you want.
    Paste - Merge Formatting
    		Content will take on the formatting on your document where you have placed the insertion point.
    Paste - Picture
    		Pastes as an image. Useful if you need a snapshot of content that you don't want altered.
    Paste - Keep Text Only
    		Pastes text without any formatting it originally had. This is especially useful when copying text from elsewhere like another document or a webpage.

    Format Painter

    This is a useful tool which applies formatting from selected formatted text to other text. This is much faster to apply the same custom formatting elsewhere.

    Note: For headings you should use styles (explored here: Styles), the format painter can then be used to apply one heading's heading style onto other text to make that a heading.
