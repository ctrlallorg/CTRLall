    Whare are indents?

    Indents are the amount of space between the margin and where a text begins. It’s a formatting setting that controls the starting position of one or more lines in a paragraph.

    Increasing an indent moves the entire paragraph inward away from the margin; decreasing an indent moves the paragraph closer to the margin. Indents are set using the markers on the ruler or the Paragraph settings.

    For beginners one feature to consider is the Paragraph Options which allow you to apply left and right indentation to a selected paragraph(s), see here for more: Paragraph Options - Indentation.

    Indenting with the ruler

    You can also adjust indents with indent markers on the ruler tool. You'll first need to enable to the ruler; go to View > Show and check Ruler and you'll see vertical and horizontal rulers appear on the screen.

    Make sure your insertion point is pointed on the paragraph first. On the horizontal ruler above the page you'll see three indent markers clumped together on the left like this:

    Each of these indent markers have different jobs when working with paragraphs:

    First line indent - Controls where the first line of the paragraph begins.
    Hanging Indent - Controls where all lines except the first line begin.
    Left Indent - Moves both the above indents together, shifting the entire paragraph left or right.

    Lists and indents

    When it comes to bullet and number lists, the indent markers work slightly differently. They're not aligned like in paragraphs and this is to do with the different functions they have for lists.

    First line indent - Controls where bullet/number begins.
    Hanging Indent - Controls where the text begins after the bullet.
    Left Indent - Moves both the above indents together, shifting both the bullet/number and line/paragraph left or right.

    Tab stops

    You can set custom alignment points on the horizontal ruler called tab stops that control where the insertion point jumps to when you press the Tab key. This is much better than rather than relying on Word's default half-inch jumps or default left-centre-right alignment.     The straightforward way to increase and decrease indents with paragraphs is to place the insertion point on the paragraph then select the Home > Paragraph > Increase/Decrease Indent buttons. Increasing an indent moves the paragraph inward away from the margin while decreasing the indent move it outward toward the margin.

    There are other ways to set indents as well.

    Using Tab key to indent a paragraph

    We're first going to just touch on something you may have already heard about regarding the Tab key. If you press Tab before a paragraph's first line it visually indents that first line 0.5" inward called a first-line indent. Alternatively, if you have the insertion point at the beginning of any line following the first line it will create a left indent, moving the whole paragraph over.

    If this is all you need to know then stick with that.

    A technical nerdy 'However' point about the Tab key and indents

    If you're teaching others Word, there's a slight difference between what the Tab key does and what an indent does that needs unpacking.

    The truth is the Tab key isn't designed to indent, instead what it actually does is move the insertion point to the next default tab stop, which sits every 0.5" across the ruler. See more on tab stops here: Tab stops

    So what's going on?

    When you press Tab it moves the insertion point the next default tab stop. However at the same time Word’s default settings automatically apply the same shift to the paragraph’s first‑line indent marker (or if your insertion point is left of a line below the first line, the whole left indent marker moves, moving the whole paragraph right.)

    So the Tab key doesn't have anything to do with indents, it's Word’s automatic behaviour that changes the text's indentation. Essentially, Word blends the two features:

    • Tab: moves the insertion point to a tab stop
    • Indent: paragraph formatting
