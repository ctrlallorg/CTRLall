<section class="section-light">

<p>Text and other content in Word can be aligned to be left, right or centre but there are a few more things you can do with alignment using something called <strong>tab stops</strong>. More advanced featured with tab stops are explored in the <a href="?difficulty=intermediate">intermediate version</a> of this article, but here we'll stick with the basics.</p>

<h2>What is a tab stop?</h2>
<p>A tab stop is a horizontal location on the page you can move the <span class="glossary-hover" data-term="insertion-point">insertion point</span> to; this lets you add text or content from that point on the page. Word has default tab stops spaced at 0.5 inches apart but you can also create custom ones of your own.</p>

<h2>How to insert a tab stop</h2>
<p>Let's focus on the default Word tab stops first. These are easy, you can add a default tab stop by pressing the <strong>Tab</strong> key:</p>
<p><img class="left" src="{{ '/assets/images/keyboard shortcuts/Tab.png' | url }}" alt="Tab key" style="max-width:125px; width:100%; height:auto; vertical-align:middle"></p>

<p>This moves the <span class="glossary-hover" data-term="insertion-point">insertion point</span> 0.5 inches to the right. So if you're at 0", pressing <strong>Tab</strong> jumps to 0.5". Press it again and it jumps to 1.0".</p>
<img class="thumbnail border" src="{{ '/assets/images/word/Tab stops/Default tab stop on page.png' | url }}" alt="Default tab stop on page in Word" style="max-width:400px; width:100%; height:auto; vertical-align:middle">

<p>From here you can continue to type further away from the margin. Tab across as far as you need to then type.</p>

<h2>Custom tab stops</h2>
<p>If you want to set the alignment yourself rather than be locked to Word's default tab stops then you can do this by inserting your own tab stop on the horizontal ruler above the page.</p>
<p>To do this simply <strong>left click</strong> once on the horizontal ruler where you want text to start from. You'll see two things happen.</p>
<ol>
    <li><p>A small grey 'L' shape will appear on the bottom of the ruler where you clicked:</p>
    <p><img class="thumbnail left" src="{{ '/assets/images/word/Tab stops/Tab stop left.png' | url }}" alt="Left-alighed tab stop symbol in Word" style="max-width:50px; width:100%; height:auto; vertical-align:middle"></p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Tab stops/Tab stop left on ruler.png' | url }}" alt="Left-alighed tab stop symbol on ruler in Word" style="max-width:150px; width:100%; height:auto; vertical-align:middle"></p></li>
    <li><p>The <span class="glossary-hover" data-term="insertion-point">insertion point</span> jumps to that point accordingly. Now text will type left aligned from where you've inserted the tab stop:</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Tab stops/Tab stop left on page.png' | url }}" alt="Left-alighed text aligned by custom tab stop on ruler in Word" style="max-width:450px; width:100%; height:auto; vertical-align:middle"></p>
    </li>
</ol>


<p>You can do more with Tab stops like change their alignment and custom spacing for specific measurement. This is all explored in the <a href="?difficulty=intermediate">intermediate version</a> of this article.</p>

<h2 id="tab-stops">Tab stops</h2>
<p>You can set custom points on the horizontal ruler called <strong>tab stops</strong> that control where the insertion point jumps to when you press the Tab key. This is useful if you don't want to rey on Word's default half-inch jumps, as you can create consistent alignment in your document how you want it and where you want it.</p>
<br>
<p><i>Default tab stop — every 0.5" by default. <u>Note</u>: you can change this, see <a href="#tabs-options">Tabs options</a> further down.</i></p>
<img class="thumbnail border left" src="{{ '/assets/images/word/Tab stops/Default tab stop on page.png' | url }}" alt="Default tab stop on page in Word" style="max-width:350px; width:100%; height:auto; vertical-align:middle">
<p><i>Custom tab stops — you choose the alignment position</i></p>
<p><img class="thumbnail border left" src="{{ '/assets/images/word/Tab stops/Multiple tab stops example.png' | url }}" alt="Multiple tab stops example in Word" style="display: inline; max-width:600px; width:100%; height:auto; vertical-align:middle"></p>

<p>Tab stops are left-aligned by default but you can also change the alignment from left to centre, right, decimal, or bar via <strong>Tabs options</strong>. These are the tab stop symbols for all of them. Examples of each are shown further below.</p> 
<div class="image-row">

<p>You can also customise tabs to have <strong>leaders</strong>. A <span class="glossary-hover" data-term="leader">leader</span> is a filler pattern (dots, dashes, or a solid line) that appears between text and the tab stop. When you press Tab, Word will insert the leader automatically. You'll recognise leaders from tables of contents but they're also using in forms, menus and other lists. You can add a leader to a tab stop in the <strong>Tabs options</strong>.</p>
<p><img class="thumbnail border" src="{{ '/assets/images/word/Tab stops/Leader example.png' | url }}" alt="Leader example in Word" style="max-width:500px; width:100%; height:auto; vertical-align:middle"></p>


<h2 id="tabs-options">Tabs options</h2>
<p>To access the Tabs options just double click on a tab stop icon.</p> 

<p>Alternatively you can navigate to the Tabs options via <strong>Home</strong> > <strong>Paragraph options</strong> (bottom right button) > <strong>Tabs</strong>.</p>
<p><img class="thumbnail border" src="{{ '/assets/images/word/Tab stops/Tab options alternative navgiation.png' | url }}" alt="Tab options alternative navgiation in Word" style="display: inline; max-width:600px; width:100%; height:auto; vertical-align:middle"></p>

<p>The Tabs options dialog shows all the tab stops you have currently, their position measurement, alignment and any leaders you have on them. Hover over each for an explanation.</p>

<h4>Bar</h4>
<p><strong>Bar</strong> is slightly different. It doesn't align text. Instead it creates a decorative vertical line at the tab position that acts as a visual divider. This might be used to separate text in headers and footers (Document title | Page number) or forms (Name | Phone | Email), in multi‑column lists without tables like a menu or directory, or anywhere you want a vertical line but don’t want the overhead of a table or shape.</li>

document.querySelectorAll('.rich-tooltip-trigger').forEach(trigger => {
  const id = trigger.dataset.tooltipId;
  const layer = document.getElementById('tooltip-layer');

  trigger.addEventListener('mouseenter', () => {
    if (!tooltips[id]) return;

    // clear previous
    layer.innerHTML = '';

    // build tooltip
    const wrapper = document.createElement('div');
    wrapper.innerHTML = tooltips[id];
    const tooltip = wrapper.firstElementChild;

    // position tooltip near hotspot
    const rect = trigger.getBoundingClientRect();
    tooltip.style.position = 'fixed';
    tooltip.style.top = `${rect.top + (rect.height / 2)}px`;
    tooltip.style.left = `${rect.right + 10}px`;
    tooltip.style.display = 'block';

    // rely on CSS for max-width and wrapping
    tooltip.style.maxWidth = "400px";

    // add to layer
    layer.appendChild(tooltip);

    // keep visible while hovering tooltip
    tooltip.addEventListener('mouseenter', () => {
      tooltip.classList.add('visible');
    });
    tooltip.addEventListener('mouseleave', () => {
      tooltip.remove();
    });
  });

  trigger.addEventListener('mouseleave', () => {
    const tooltip = document.querySelector('#tooltip-layer .rich-tooltip');
    if (tooltip) {
      // delay so you can move into tooltip
      setTimeout(() => {
        if (!tooltip.matches(':hover')) {
          tooltip.remove();
        }
      }, 150);
    }
  });
});
