  <section class="section-light">
  <h2>Find</h2>
  <p>You can search for words or phrases in a document with the <strong>Find</strong> function which is found on the far right of Home tab on the <span class="glossary-hover" data-term="ribbon">ribbon</span>:</p>
  <img class="thumbnail border left" src="{{ '/assets/images/word/Find/Find on ribbon.png' | url }}" alt="Find option on ribbon in word" style="max-width:138px; width:100%; height:auto; vertical-align:middle">
  <p>Alternatively you can also use the keyboard shortcut <strong>Ctrl + F</strong>:</p>
  <img class="thumbnail border left" src="{{ '/assets/images/keyboard shortcuts/Ctrl+F.png' | url }}" alt="Find function Keyboard shortcut Ctrl and F" style="max-width:200px; width:100%; height:auto; vertical-align:middle">
  <p>This will bring up the navigation pane where you can type in a word or phrase in the text box. Word will show matching entries below in the navigation pane and highlight the matching word(s) on the page throughout the document. The navigation pane can be moved by dragging it, you can also dock it to the right side of the screen.</p>
  <img class="thumbnail border" src="{{ '/assets/images/word/Find/Results search keyword.png' | url }}" alt="Example searching for a keyword via the navigation pane in Word" style="max-width:900px; width:100%; height:auto; vertical-align:middle">
  <p>You can cycle through the matching results by pressing <strong>Enter</strong> or clicking the <strong>arrow buttons</strong> below.</p>
   <img class="thumbnail border" src="{{ '/assets/images/word/Find/Cycle through results.gif' | url }}" alt="Arrow buttons to cycle through search results on navigation pane in Word" style="max-width:900px; width:100%; height:auto; vertical-align:middle">
  <p>The navigation pane has three tabs showing the location of the match:</p>
  <ul>
    <li><strong>Headings</strong>: Indicates what headings or subheadings the results belong to, this can be useful to locate specific text within a large or complex document with several sections.</li>
    <li><strong>Pages</strong>: Gives a preview of pages where the matching results are located. If the document pages are easily discernable then this method may be of use.</li>
    <li><strong>Results</strong>: Shown by default, will list the individual matching results and surrounding text which provides a short snapshot of context. In most cases this is the most useful.</li>
  <div class="image-row">
    <img src="{{ '/assets/images/word/Find/Headings search results.png' | url }}" alt="Navigation pane find search results under Headings tab" style="max-width:300px; width:100%; height:auto; vertical-align:middle">
    <img src="{{ '/assets/images/word/Find/Pages search results.png' | url }}" alt="Navigation pane find search results under Pages tab" style="max-width:300px; width:100%; height:auto; vertical-align:middle">

  </div>

  <h2>Advanced Find</h2>
  <p>Advanced Find is a useful tool if you're looking for a specific phrase with formatting, case, special symbol characters and gives you the option to use wildcards. It is found on the Home tab of the ribbon by clicking the drop down menu of the <strong>Find</strong> option (i.e. the arrow to the right of the Find button), and then selecting <strong>Advanced Find</strong>.</p>
  <p><img class="thumbnail border left" src="{{ '/assets/images/word/Find/Advanced Find on ribbon.png' | url }}" alt="Advanced Find option on ribbon in Word" style="max-width:195px; width:100%; height:auto; vertical-align:middle"></p>
  <img src="{{ '/assets/images/word/Find/Advanced Find.png' | url }}" alt="Advanced Find dialog" style="max-width:560px; width:100%; height:auto; vertical-align:middle">

  <h3 id="find-and-replace">Find and Replace</h3>
  <p>There is also the option to <strong>Find and Replace</strong> where you can search characters, words or terms and replace them individually (via <strong>Replace</strong>) or all instances throughout the document (via <strong>Replace All</strong>). The keyboard shortcut to this feature is <strong>Ctrl + H</strong>.</p>
  <img class="thumbnail left" src="{{ '/assets/images/keyboard shortcuts/Ctrl+H.png' | url }}"  alt="Keyboard shortcut for Find and Replace Ctrl+H" style="max-width:200px; width:100%; height:auto; vertical-align:middle">
  <img src="{{ '/assets/images/word/Find/Find and replace.png' | url }}" alt="Advanced Find dialog" style="max-width:560px; width:100%; height:auto; vertical-align:middle">
  <p>Find and replace can be useful for mass formatting and correction of terms and even punctuation, e.g. changing all double quotation marks (&ldquo;&rdquo;) to single quotation marks (&lsquo;&rsquo;) in a document if you needed to, or <a href="/word/fixing-irregular-spacing">fixing irregular line spacing</a>.</p>

  <h4>Match case</h4>
  <p>Make the search case-sensitive.</p>

  <h4>Match whole words only</h4>
  <p>Only matches your search term when it appears as a complete word, surrounded by spaces or punctuation. The matching word must be in standalone form. For example, if you have the word 'caption' in your document, searching for 'tion' with <em>Match whole words only</em> enabled won't pick up 'caption'.</p>

  <h4>Use wildcards</h4>
  <p>Wildcards are pattern characters that let you build flexible search patterns such as * for any string of characters and ? for any single character. This is an advanced feature which is explored with the use of <span class="glossary-hover" data-term="find-and-replace">Find and Replace</span> in the <a href="/word/fixing-irregular-spacing/">Fixing irregular spacing</a> article.</p>
  <p>Enabling wildcards disables <em>Match case</em>, <em>Match whole words only</em>, <em>Match prefixes</em> and <em>Match suffixes</em>.</p>  

  <h4>Sounds like</h4>
  <p>Finds words phonetically similar to your search term. This feature is limited to English and doesn't always produce reliable results. Best avoided.</p>

  <h4>Find all word forms</h4>
  <p>This matches grammatical variations of a search word. It works for variations of verbs, e.g. if you search 'ran' with <em>Find all word forms</em> enabled Word would find results matching like 'run', 'runs' and 'running'. It also works with some nouns with singular and plural variations.</p>
  <p><u>Note</u>: This option doesn't work with wildcards enabled.</p>

    tooltip.style.maxWidth = "400px";

    layer.appendChild(tooltip);

    // keep visible while hovering tooltip
    tooltip.addEventListener('mouseenter', () => {
      tooltip.classList.add('visible');
    });
    tooltip.addEventListener('mouseleave', () => {
      tooltip.remove();
    });
  });

  trigger.addEventListener('mouseleave', () => {
    const tooltip = document.querySelector('#tooltip-layer .rich-tooltip');

          tooltip.remove();
        }
      }, 150);
    }
  });
