<p>You can format your text into lists easily in Word. The main two types are bullet lists and numbered lists. Here's how to do both.</p>

<h3>Bullet lists</h3>
<p>You might hear this referred to as 'bulleted lists' or just 'bullets'. These lists are useful for itemised lists in no order.</p>

<br>
<p>Shopping list:</p>
<ul>
  <li>Milk</li>
  <li>Bread</li>
  <li>Eggs</li>
  <li>Fruit</li>
</ul>
<br>

<h3>Numbered lists</h3>
<p>These are sometimes referred to as 'Number lists' as well. These lists are useful for steps, processes and sequences that run in a certain order or for prioritised lists.</p>
<br>
<p><u>Sequential</u></p>
<p>Basic workflow:</p>
<ol>
  <li>Open the file.</li>
  <li>Make your edits.</li>
  <li>Save the document.</li>
</ol>
<br>
<p><u>Priority</u></p>
<p>Work priorities:</p>
<ol>
  <li>Complete urgent tasks.</li>
  <li>Reply to important messages.</li>
  <li>Update project notes.</li>
</ol>

<br>
<p>So how do you do these in Word?</p>

<h3>Starting a list in Word</h3>
<p>Click the buttons in the <strong>Home > Paragraph</strong> section on the ribbon:</p>
 <p><img class="thumbnail border" src="{{ '/assets/images/word/Lists/Bullet and number list buttons on ribbon.png' | url }}" alt="Bullet list and number list buttons on ribbon in Word" style="max-width:600px; width:100%; height:auto; vertical-align:middle"></p>
<p>The first bullet or number list point is created ready for you to type in. Type in text as needed then when ready to add the next point in hit <strong>Enter</strong> and a new bullet/number will be created.</p>
<p>To end the list hit <strong>Enter</strong> a second time or click the button to go back to a normal paragraph.</p>
<br>
<i>Bullet list</i>
<p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Bullet list.gif' | url }}" alt="Bullet list demonstration in Word" style="max-width:350px; width:100%; height:auto; vertical-align:middle"></p>

<i>Numbered list</i>
<p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Numbered list.gif' | url }}" alt="Numbered list demonstration in Word" style="max-width:350px; width:100%; height:auto; vertical-align:middle"></p>

<h3 id="format-text-into-list">Format text already typed into a list</h4>
<p>If you want to convert existing text into a list: 
<ol>
  <li>  <p>First select the lines you want to make into a list. (For more on how to select text in different ways see this article: <a href="/word/selecting-text/">Selecting text</a>)</p>
      <p><u>Note</u>: Each list entry needs to start on a new paragraph line; this is done by applying a <span class="glossary-hover" data-term="hard-return">hard return</span> after each entry — press <strong>Enter</strong> after each entry to push the <span class="glossary-hover" data-term="insertion-point">insertion point</span> down to the next paragraph line.</p>
  </li>
  <li><p>Then click on of the two options above for a bullet list or numbered list</p>
  <p><img class="thumbnail border" src="{{ '/assets/images/word/Lists/Bullet and number list buttons on ribbon.png' | url }}" alt="Bullet list and number list buttons on ribbon in Word" style="max-width:600px; width:100%; height:auto; vertical-align:middle"></p></li>
</ol>

<p>You may have some experience with bullets and number lists already, creating them and applying them to existing text via the two button found on the ribbon under <strong>Home > Paragraph</strong>.</p>

<p><img class="thumbnail border" src="{{ '/assets/images/word/Lists/Bullet and number list buttons on ribbon.png' | url }}" alt="Bullet list and number list buttons on ribbon in Word" style="max-width:600px; width:100%; height:auto; vertical-align:middle"></p>

<p>But you can do more.</p>

<h2>List styles</h2>
<p>You don't have to settle for the default black dots for bullets and (1. 2. 3.) for numbered lists. In some documents you may require bullets that looks more formal or roman numerals (i, ii, iii.). Here's how to apply other styles.</p>
<p><u>Note</u>: You can also create your own <a href="#custom-styles">custom styles</a>.</p>


<h3>Bullet styles</h3>
<ol>
  <li>  
    <p>If you're starting a new bullet list skip this step and go to step 2.</p>
    <p>If you already have a bulleted list, first click on one of the bullets and all the bullets at that level will be selected like this:</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Bullets selected.png' | url }}" alt="Bullets selected in Word" style="max-width:150px; width:100%; height:auto; vertical-align:middle"></p>
    <p>If you have text that isn't bulleted yet, select all the text (remember each entry needs to be separated by a <span class="glossary-hover" data-term="hard-return">hard return</span> to start on a new paragraph line; press <strong>Enter</strong> after each list entry).</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Text for bullets selected.png' | url }}" alt=" Text for bullets selected in Word" style="max-width:150px; width:100%; height:auto; vertical-align:middle"></p>
  </li>
  <li> 
    <p>Click the arrow button right of the <strong>Bullets</strong> button on the ribbon under <strong>Home > Paragraph</strong>. This will bring up a drop down menu with a <strong>Bullet Library</strong> with different styles to choose from. Select the one you want and it will be applied for the list.</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Bullet library.png' | url }}" alt="Bullet library options in Word" style="max-width:300px; width:100%; height:auto; vertical-align:middle"></p>
  </li>
</ol>

<p>If you don't like any of those options you can also created your own custom bullets by making a <a href="#custom-styles">custom bullet style</a>.</p>

<h3>Numbering styles</h3>
<ol>
  <li>  
    <p>If you're starting a new numbered list skip this step and go to step 2.</p>
    <p>If you already have a numbered list, first click on one of the numbers themselves and all the numbers at that level will be selected like this:</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Numbering selected.png' | url }}" alt="Numbering selected in a list in Word" style="max-width:200px; width:100%; height:auto; vertical-align:middle"></p>
    <p>If you have text that isn't in a numbered list yet, select all the text (remember each entry needs to be separated by a <span class="glossary-hover" data-term="hard-return">hard return</span> to start on a new paragraph line; press <strong>Enter</strong> after each list entry).</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Text for numbering selected.png' | url }}" alt=" Text for bullets selected in Word" style="max-width:200px; width:100%; height:auto; vertical-align:middle"></p>
  </li>
  <li> 
    <p>Click the arrow button right of the <strong>Numbering</strong> button on the ribbon under <strong>Home > Paragraph</strong>. This will bring up a drop down menu with a <strong>Numbering Library</strong> with different styles to choose from. Select the one you want and it will be applied for the list.</p>
    <p><img class="thumbnail border left" src="{{ '/assets/images/word/Lists/Numbering library.png' | url }}" alt="Bullet library options in Word" style="max-width:300px; width:100%; height:auto; vertical-align:middle"></p>
  </li>
</ol>

<p>If you prefer to make your own style see the next section.</p>

<h2 id="custom-styles">Custom styles</h2>
<p>Customise your own styled list for a bullet list or numbered list.</p>

<h3>Custom bullets</h3>
<p>Click the arrow button right of the <strong>Bullets</strong> button on the ribbon under Home > Paragraph and at the bottom of the drop down menu select <strong>Define new bullet</strong>.</p>

<p>This brings up a dialog with options to switch the bullet to a symbol, image or text. The atter two are less common. To use symbol select the <strong>Symbol</strong> button and options to select from are shown in a new dialog. Arrows are normally a good alternative.</p>
<p><img class="thumbnail" src="{{ '/assets/images/word/Lists/Define new bullet.png' | url }}" alt="Define new bullet dilog in Word" style="max-width:750px; width:100%; height:auto; vertical-align:middle"></p>

<p><img class="thumbnail" src="{{ '/assets/images/word/Lists/Custom bullet example 0.png' | url }}" alt="Custom bullet selection in Word" style="max-width:750px; width:100%; height:auto; vertical-align:middle"></p>

document.querySelectorAll('.rich-tooltip-trigger').forEach(trigger => {
  const id = trigger.dataset.tooltipId;
  const layer = document.getElementById('tooltip-layer');

  trigger.addEventListener('mouseenter', () => {
    if (!tooltips[id]) return;

    // clear previous
    layer.innerHTML = '';

    // build tooltip
    const wrapper = document.createElement('div');
    wrapper.innerHTML = tooltips[id];
    const tooltip = wrapper.firstElementChild;

    // position tooltip near hotspot
    const rect = trigger.getBoundingClientRect();
    tooltip.style.position = 'fixed';
    tooltip.style.top = `${rect.top + (rect.height / 2)}px`;
    tooltip.style.left = `${rect.right + 10}px`;
    tooltip.style.display = 'block';

    // rely on CSS for max-width and wrapping
    tooltip.style.maxWidth = "400px";

    // add to layer
    layer.appendChild(tooltip);

    // keep visible while hovering tooltip
    tooltip.addEventListener('mouseenter', () => {
      tooltip.classList.add('visible');
    });
    tooltip.addEventListener('mouseleave', () => {
      tooltip.remove();
    });
  });

  trigger.addEventListener('mouseleave', () => {
    const tooltip = document.querySelector('#tooltip-layer .rich-tooltip');
    if (tooltip) {
      // delay so you can move into tooltip
      setTimeout(() => {
        if (!tooltip.matches(':hover')) {
          tooltip.remove();
        }
      }, 150);
    }
  });
});
</script>
