  <h2 id="object-text-box-beg">Text boxes</h2>
  <p>Text boxes are frames that can contain text and other content separately from the main document page. You might want to use a text box for a side note, label, caption or just custom layout.</p>
  <p>To add a text box go to <strong>Insert > Text box</strong>. Here you have a series of templated text boxes to choose from. The standard one is top left.</p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Text box menu.png' | url }}" alt="Text Box drop down menu on ribbon in Word"  style="max-width:400px; width:100%; height:auto; vertical-align:middle"></p>
  <p>This place a text box on the page with some placeholder text which you can edit as needed. You can freely move, resize and rotate the text box as well. You can also insert images into a text box, it's like a minature page within the <span class="glossary-hover" data-term="selection-frame">selection frame</span>.</p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Text box with placeholder text.png' | url }}" alt="Text Box inserted with placeholder text in Word"  style="max-width:250px; width:100%; height:auto; vertical-align:middle"></p>
  <h3>Draw text box</h3>
  Draw your own text box to create a custom blank one. To do this go to <strong>Insert > Text box > Draw Text Box</strong>. Another way that's slightly quicker is via the Shapes menu; go to <strong>Insert > Shapes</strong> and click the icon with the rectangle with the 'A'.</p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Text box via shapes.png' | url }}" alt="Text Box option on Shapes drop down menu on ribbon in Word"  style="max-width:300px; width:100%; height:auto; vertical-align:middle"></p>
  <p>Now your cursor will change to a cross hair and you can click and drag out a blank text box to the size you want.</p>
  <p><img class="thumbnail left" src="{{ '/assets/images/word/Objects/Draw text box cursor crosshair.png' | url }}" alt="Draw text box crosshair cursor in Word" style="max-width:50px; width:100%; height:auto; vertical-align:middle"></p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Draw text box.gif' | url }}" alt="Text Box option on Shapes drop down menu on ribbon in Word"  style="max-width:600px; width:100%; height:auto; vertical-align:middle"></p>

  <p>On the ribbon the <strong>Equation</strong> tab provides tools to build an equation.</p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/equation tab ribbon.png' | url }}" alt="Text Box option on Shapes drop down menu on ribbon in Word"  style="max-width:800px; width:100%; height:auto; vertical-align:middle"></p>

This is explored in the <a href="?difficulty=intermediate">intermediate version</a> of this article.</li>

  <h2 id="object-text-box">Inserting text boxes</h2>
  <p>Text boxes are exactly what they sound like. When you insert a text box onto the page you can enter text into it, change the size, position, style and format of the text box as needed. You can also insert images into them.</p>
  <p>The quickest way to insert a text box is via <strong>Insert > Shapes</strong> - it's the top left option under the <strong>Basic shapes</strong> category. There are templated text boxes you can use too which are found under the <strong>Insert > Text boxes</strong> menu on the ribbon.</p>
  <p>Text boxes are adjustable, formattable and wrappable; to see how to alter their properties, look at this article: <a href="/word/altering-images-and-objects/">Altering images and objects</a>.</p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Text box via shapes.png' | url }}" alt="Text box option in the shapes menu in Word"  style="max-width:300px; width:100%; height:auto; vertical-align:middle">></p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Text box menu.png' | url }}" alt="Text Box drop down menu on ribbon in Word"  style="max-width:400px; width:100%; height:auto; vertical-align:middle"></p>


  <h2 id="object-equation">Inserting equations</h2>
  <p>Word offers some pre-made equations from the <strong>Equations</strong> drop down menu but you'll likely need to create one from scratch, so click the top half of the equations button, not the drop down.</p>
  <p>An equations field will appear on the page and the <strong>Equations</strong> tab will now be visible on the ribbon with tools to create the equation. See more on equations here: <a href="/word/equations/">Equations</a></p> 
  <p><img class="thumbnail left" src="{{ '/assets/images/word/Objects/Equation menu.png' | url }}" alt="Equation option on ribbon in word"  style="max-width:115px; width:100%; height:auto; vertical-align:middle"></p>
  <p><img class="thumbnail left" src="{{ '/assets/images/word/Objects/Equation field.png' | url }}" alt="Equation field inserted on page"  style="max-width:200px; width:100%; height:auto; vertical-align:middle"></p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Equation tab ribbon.png' | url }}" alt="Equation tab options on ribbon in Word"  style="max-width:800px; width:100%; height:auto; vertical-align:middle"></p>

  <h2 id="object-embedded-files">Inserting embedded files (OLE objects)</h2>
  <p>You can embed existing files onto a page of a document. This acts as a way of attaching file content to your Word document either as a static standalone copy of that file or linked to the source file. There's a lot to unpack about these, to understand everything about them look at this article: <a href="/word/objects-embedded-files/">Embedded files (OLE objects)</a>.</p>
  <p><img class="thumbnail border" src="{{ '/assets/images/word/Objects Embedded files/OLE objects.png' | url }}" alt="Embedded files inserted into Word" style="max-width:600px; width:100%; height:auto; vertical-align:middle"></p>


  <h2 id="object-drawing-canvas">Inserting drawing canvases</h2>
  <p>A drawing canvas is a little-known container that groups images, graphic visuals and text boxes together in a locked area, which is useful if you need certain content to remain together in one space.</p>
  <p>To insert a drawing canvas go to <strong>Shapes</strong> and select <strong>New Drawing Canvas</strong> at the very bottom of the drop down menu.</p>
  <p>Once inserted the canvas itself behaves like a single object, it's formattable and wrappable, but while selected you can add images, other graphic visuals and text boxes inside it. See more on how to do this here:</p>
  <p><img class="thumbnail" src="{{ '/assets/images/word/Objects/Drawing canvas menu.png' | url }}" alt="Drawing canvas option from Shapes menu on ribbon in Word" style="max-width:300px; width:100%; height:auto; vertical-align:middle"></p>
  <p><img class="thumbnail border" src="{{ '/assets/images/word/Objects/Drawing canvas example.gif' | url }}" alt="Demonstration of object moved in a drawing canvas" style="max-width:650px; width:100%; height:auto; vertical-align:middle"></p>
