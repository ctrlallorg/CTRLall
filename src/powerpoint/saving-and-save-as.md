<h1>Saving and Save As</h1>

<article class="difficulty-wrapper" role="article" aria-label="Altering images tutorial with beginner and intermediate versions">
<div id="article-badge" class="article-badge beginner">⭐ Beginner Level</div>
<!-- Beginner version -->
<div class="difficulty beginner-version"  aria-label="Beginner introduction to altering images">

  <section class="section-light">
  
  <p>Saving your PowerPoint presentation lets you to keep your edits for next time. When you save, the next time you open your presentation those changes you made will be rememebered. Not saving means you will lose all the edits made last time. Here's how to save a PowerPoint presentation:</p>
  
  <p>With the presentation open, you can save by doing one of the following:</p>
  <ol type="a">
  <li><strong>Save the existing presentation</strong> (<u>Note</u>: if it has already been saved previously you ‘overwrite’ the file with changes made). Either press <strong>Ctrl + S</strong> on your keyboard or click the <strong>Save</strong> button at the top left of the <span class="glossary-hover" data-term="quick-access-toolbar">Quick Access Toolbar</span>.</p>

  <img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save As/Save on QAT.png' | url }}"  alt="Save button on quick access toolbar in PowerPoint" style="max-width:225px; width:100%; height:auto; vertical-align:middle">

  </li>

  <p><strong>OR</strong></p>

  <li><strong>Save as a new presentation or a different <span class="glossary-hover" data-term="filetype" style="font-weight:bold;">file type</span></strong> (e.g. a template or PDF) using <strong>File > Save As.</strong></li>

  Select <strong>File</strong>
  <img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/File.png' | url }}"  alt="File tab button on ribbon in PowerPoint" style="max-width:175px; width:100%; height:auto; vertical-align:middle">
  Then <strong>Save As</strong></p>
  <img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/File Save As.png' | url }}"  alt="Save as button under file in PowerPoint" style="max-width:200px; width:100%; height:auto; vertical-align:middle">

  <p>This will then bring up options to choose where to save your PowerPoint presentation, what the name of the file should be, and what file type to save it as. Double click <strong>This PC</strong> to browse a location.</p>
  <p><img class="thumbnail border left" src="{{ '/assets/images/powerpoint/Saving and Save as/File Save As 2.png' | url }}"  alt="Save As options This PC in PowerPoint" style="max-width:900px; width:100%; height:auto; vertical-align:middle"></p>

  A window appears which lets you decide on the <strong>file name</strong>, the <strong>file type</strong> (the dropdown menu gives a large list of options) and the <strong>location</strong> to save it.
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/Browse window.png' | url }}"  alt="Save As browse window in PowerPoint" style="max-width:700px; width:100%; height:auto; vertical-align:middle"></p>
  </ol>

  <h2>Autosave</h2>
  <p>If you're using Microsoft Office 365 you'll see an extra <strong>Autosave</strong> toggle at the top left of the screen on the <span class="glossary-hover" data-term="quick-access-toolbar">Quick Access Toolbar</span>. This is for when your files are stored on <strong>OneDrive</strong> or <strong>SharePoint</strong>. Autosave automatically saves your edits every few seconds, so you don't need to manually save.</p>
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/Autosave.png' | url }}"  alt="Autosave toggle in PowerPoint" style="max-width:200px; width:100%; height:auto; vertical-align:middle"></p>

  <p>If you're using PowerPoint on your mobile or the free version PowerPoint Online then your edits automatically save too, there just isn't a toggle to turn autosaving off.</p>
  </section>

  </li>

  <li><strong>Save as a new presentation or a different <span class="glossary-hover" data-term="filetype" style="font-weight:bold;">file type</span></strong> (e.g. a template or PDF) using <strong>File > Save As.</strong></li>

  Select <strong>File</strong>
  <img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/File.png' | url }}"  alt="File tab button on ribbon in PowerPoint" style="max-width:175px; width:100%; height:auto; vertical-align:middle">
  Then <strong>Save As</strong></p>

  <p>This will then bring up these options below to choose where to save, what the name of the file should be and what file type you need to save it as. Double click <strong>This PC</strong> to browse a location.</p>

  <p><img class="thumbnail border left" src="{{ '/assets/images/powerpoint/Saving and Save as/File Save As 2.png' | url }}"  alt="Save As options This PC in PowerPoint" style="max-width:900px; width:100%; height:auto; vertical-align:middle"></p>

In the example below the location field says 'OneDrive - Personal'.</p>
  In the browser window that appears decide on the file name, the file type (the dropdown menu gives a large list of options) and the location to save it.

  <h2>Autosave</h2>
  <p>If  are using PowerPoint as part of Microsoft Office 365 you'll see an <strong>Autosave</strong> toggle found at the top left of the screen. This is for when your files are stored on <strong>OneDrive</strong> or <strong>SharePoint</strong>. Autosave automatically saves your edits every few seconds, so you don't need to manually save.</p>
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/Autosave.png' | url }}"  alt="Autosave toggle in PowerPoint" style="max-width:200px; width:100%; height:auto; vertical-align:middle"></p>

  <p>If you're using PowerPoint on your mobile or the free version PowerPoint Online then your edits automatically save too, there just isn't a toggle to turn autosaving off.</p>

  <h4>'Autosave isn't working'</h4>
  <p>Take a look at this article: <a href="/all/autosave-not-working">Autosave not working</a></p>

  <h2 id="default-save-location">Default save locations</h2>
  <p>You can change the folder PowerPoint default saves to. This will be the first folder that appears in the <i>Save As</i> browsing window; you might want to change this to somewhere else such as your desktop. To change the default save location go to <strong>File</strong> top left, then on the next screen select <strong>Options</strong> (you might need to click <strong>More</strong> to see <strong>Options</strong>):</p>
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/File.png' | url }}"  alt="File tab button on ribbon in PowerPoint" style="max-width:175px; width:100%; height:auto; vertical-align:middle"></p>
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/File Options.png' | url }}"  alt="Save As browse window in PowerPoint" style="max-width:600px; width:100%; height:auto; vertical-align:middle"></p>

  <p>In the PowerPoint Options dialog select <strong>Save</strong> from the left menu:</p>
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/Options save.png' | url }}"  alt="Save As browse window in PowerPoint" style="max-width:280px; width:100%; height:auto; vertical-align:middle"></p>

  <p>With the Save Options you'll see two fields for default locations, one for presentations and one for templates, copy in the folder path you want and click <strong>OK</strong> to change it.</p>
  <p><img class="thumbnail left" src="{{ '/assets/images/powerpoint/Saving and Save as/Options Save Options.png' | url }}"  alt="Save As browse window in PowerPoint" style="max-width:800px; width:100%; height:auto; vertical-align:middle"></p>
